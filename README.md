A web-based Library Management System developed using Java Servlets, JDBC, and HTML.
This project allows administrators to add, view, and manage books and authors stored in a database.

FEATURES:
Add new books to the library
View available books
Manage author information
Database connectivity using JDBC
Simple UI using HTML pages
MVC-based structure (Servlet → Service → DAO → DB)

TECHNOLOGIES:
Java
Java Servlets
JDBC
HTML
Oracle Database
Apache Tomcat Server
Eclipse IDE
